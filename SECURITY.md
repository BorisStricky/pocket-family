# Security Policy

## Supported Versions

Only the latest version on the `master` branch is actively maintained.

| Version | Supported |
|---------|-----------|
| latest (master) | ✅ |
| older branches | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability, please [open a GitHub Issue](https://github.com/BorisStricky/pocket-family/issues) with the label `security`.

For sensitive issues, describe the vulnerability in general terms in the issue title and request to discuss details privately in the comments.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We aim to respond within 5 business days.
